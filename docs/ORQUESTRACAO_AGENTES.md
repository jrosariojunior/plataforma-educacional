# 🎼 Orquestração de Agentes — Equipe Completa

## Visão Geral da Equipe

A **Equipe de Agentes para Matemática do Prof. José Rosário** é um sistema integrado de 5 agentes especializados que cobrem todas as necessidades de produção de conteúdo didático para aulas de matemática (Fundamental e Médio).

```
                    🎓 PROFESSOR JOSÉ ROSÁRIO
                             ↓
        ┌────────────────────────────────────────────┐
        │     EQUIPE DE 5 AGENTES ESPECIALIZADOS     │
        └────────────────────────────────────────────┘
              ↓                ↓               ↓               ↓
        ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
        │ Avaliador│    │Problemista│    │Sintetizar│    │ Diferenc.│
        │   Mat    │    │   Mat     │    │          │    │   Mat    │
        │ PROVAS   │    │EXERCÍCIOS │    │ RESUMOS  │    │ATIVIDADES│
        │Dissertat.│    │Fixação    │    │Teóricos  │    │Diferenci.│
        └──────────┘    └──────────┘    └──────────┘    └──────────┘
              ↓                ↓               ↓               ↓
        ────────────────────────────────────────────────────────────
              ↓                                                       ↓
        ┌──────────────────────────────────────┬──────────────────┐
        │        INTERATIVO MAT               │                  │
        │  Apresentações, Simuladores, Jogos  │                  │
        └──────────────────────────────────────┴──────────────────┘

## 1️⃣ Agentes e Especialidades

### AvaliadorMat — Avaliações
- **Expertise:** Provas dissertativas, critérios de Bloom, distribuição de pontos
- **Saída:** Prova + Gabarito + Verificação
- **Quando usar:** "Preciso de uma prova de..."
- **Modelos:** Rousseau OU Estado SP
- **Tópicos:** Todos (7º-9º EF, 1º-3º EM)

### ProblemistaMat — Exercícios
- **Expertise:** Listas progressivas, diferenciação por proficiência, variação de tipos
- **Saída:** Lista + Gabarito + Sugestões Pedagógicas
- **Quando usar:** "Preciso de exercícios para..."
- **Tipos:** Fixação, reforço, desafio, misto
- **Tópicos:** Todos

### SintetiZador — Resumos Teóricos
- **Expertise:** Síntese clara, estrutura hierárquica, exemplos, formulário
- **Saída:** Resumo + Formulário/Tabelas + Exemplos Resolvidos
- **Quando usar:** "Preciso de um resumo sobre..."
- **Formatos:** Textuais, com tabelas, diagramas
- **Tópicos:** Todos

### DiferenciadoMat — Atividades Diferenciadas
- **Expertise:** 4 versões (Abaixo do Básico, Básico, Adequado, Avançado), metodologias ativas
- **Saída:** Atividades Diferenciadas + Plano de Execução + Orientações
- **Quando usar:** "Preciso incluir alunos com ritmos diferentes..."
- **Metodologias:** Estações, rotação, pares mistos, gamificação
- **Tópicos:** Todos

### InterativoMat — Conteúdo Digital
- **Expertise:** Design instrucional, interatividade, múltiplos formatos
- **Saída:** PPTX interativo / HTML / App web + Guia + Sugestões
- **Quando usar:** "Quero uma apresentação interativa..." ou "Cria um simulador..."
- **Formatos:** PPTX, HTML/CSS/JS, Simuladores, Jogos, Quiz
- **Tópicos:** Todos

## 2️⃣ Fluxos de Uso Típicos

### Fluxo A: Preparo de Aula Completa (Semana)

```
SEGUNDA (Preparação)
    ↓
[SintetiZador] Cria resumo teórico sobre Funções Quadráticas
    ↓ (Lê o resumo para planejar)
[InterativoMat] Cria apresentação interativa para abertura de aula
    ↓
[ProblemistaMat] Cria lista de exercícios diferenciados (Fixação)
    ↓
[DiferenciadoMat] Cria rotação de estações para quinta-feira
    ↓
SEXTA (Avaliação)
    ↓
[AvaliadorMat] Cria prova bimestral sobre Funções

RESULTADO: Semana completa planejada, todos os materiais prontos!
```

### Fluxo B: Aula de Descoberta Guiada

```
OBJETIVO: Alunos descobrem propriedades de Funções Quadráticas

INÍCIO (10 min)
[InterativoMat]
→ Apresentação provocadora (gráficos dinâmicos)
→ Pergunta: "Por que a parábola muda quando mudo 'a'?"

DESENVOLVIMENTO (30 min)
[DiferenciadoMat]
→ Estação 1 (Abaixo do Básico): Conceitos básicos com suporte
→ Estação 2 (Básico): Aplicação de fórmulas
→ Estação 3 (Adequado): Análise de variações
→ Estação 4 (Avançado): Investigação aberta

FECHAMENTO (10 min)
[InterativoMat]
→ Simulador revisita descobertas
→ Síntese coletiva

CONSOLIDAÇÃO (Casa)
[ProblemistaMat]
→ Exercícios para praticar o descoberto
```

### Fluxo C: Remediação para Alunos com Dificuldades

```
DIAGNÓSTICO: Alunos não entenderam Funções Quadráticas

PASSO 1: Repassar Conceitos
[SintetiZador]
→ Resumo muito simples, com exemplos do dia a dia
→ Enfoque em pré-requisitos (EF: função linear)

PASSO 2: Oferecer Suporte Diferenciado
[DiferenciadoMat]
→ Atividades "Abaixo do Básico" com orientação 1-a-1
→ Pares mistos (aluno avançado + aluno com dificuldade)

PASSO 3: Praticar Muito
[ProblemistaMat]
→ Lista com 50% "Abaixo do Básico" + 50% "Básico"
→ Feedback imediato no gabarito

PASSO 4: Tornar Engajante
[InterativoMat]
→ Simulador para exploração (menos "pressão" que prova)
→ Gamificação (pontos por acerto)

RESULTADO: Aluno recuperado e confiante!
```

### Fluxo D: Aula Invertida (Flipped Classroom)

```
PRÉ-AULA (Casa)
1. [SintetiZador] Aluno estuda resumo sobre Matrizes
2. [InterativoMat] Assiste apresentação com simulador
3. [ProblemistaMat] Faz 5 exercícios básicos (feedback automático)

AULA (Presencial)
1. Dúvidas sobre resumo/exercícios (5 min)
2. [DiferenciadoMat] Rotação de estações para aprofundamento
3. [ProblemistaMat] Exercícios avançados em grupo
4. Discussão de resultados

PÓS-AULA
[AvaliadorMat] Prova para validar aprendizado

RESULTADO: Aula focada em aplicação, não transmissão!
```

## 4️⃣ Sequência Típica de Uma Aula Bem Estruturada

```
PREPARAÇÃO (1 semana antes)
├─ [SintetiZador] Resumo do tópico → Você lê e planeja
├─ [InterativoMat] Apresentação interativa → Para usar em aula
└─ [ProblemistaMat] Exercícios de fixação → Para os alunos

ANTES DA AULA (15 min antes)
├─ Prepare ambiente
├─ Teste apresentação interativa
└─ Imprima listas de exercícios

ABERTURA (5 min)
├─ Pergunta provocadora ou problema
├─ [InterativoMat] Simulador visivo para contextualizar
└─ Objetivo claro: "Vamos aprender..."

DESENVOLVIMENTO (30-35 min)
├─ Breve explicação (5 min)
├─ [DiferenciadoMat] Rotação de estações (25-30 min)
│  ├─ Estação 1: Alunos com dificuldade + suporte
│  ├─ Estação 2: Alunos básicos + guia
│  ├─ Estação 3: Alunos adequados + investigação
│  └─ Estação 4: Alunos avançados + desafio
└─ Você circula, faz perguntas, observa

FECHAMENTO (5-10 min)
├─ Compartilhamento rápido de cada grupo
├─ Síntese coletiva
├─ Conexão com conceito central
└─ Ponte para próxima aula

CONSOLIDAÇÃO (Casa)
├─ [ProblemistaMat] Lista de exercícios para praticar
├─ [SintetiZador] Resumo para revisar se necessário
└─ [InterativoMat] Simulador para exploração (opcional)

AVALIAÇÃO (Final do bimestre)
└─ [AvaliadorMat] Prova dissertativa

RESULTADO: Aula ativa, inclusiva, bem estruturada! 🎯
```

## 5️⃣ Integração com Skill "professor-jose"

Todos os 5 agentes usam como **base** as regras do skill `professor-jose`:

```
professor-jose (BASE)

├─ Estruturas de plano de aula
├─ Padrões de cabeçalho (Rousseau/Estado SP)
└─ Pedagogia (Bloom, metodologias ativas, BNCC)
        ↑         ↑         ↑         ↑
        └─────────┴─────────┴─────────┘
  Todos os 5 agentes respeitam essas regras!

**Relação:**
- `professor-jose`: Regras universais
- `avaliador-mat`: Especialista em provas
- `problemista-mate`: Especialista em exercícios

- `diferenciado-mat`: Especialista em inclusão
- `interativo-mat`: Especialista em interatividade

## 6️⃣ Exemplo Prático: Semana Completa de Funções Quadráticas

### SEGUNDA — Preparação
```
"@sintetizador Cria um resumo sobre Funções Quadráticas
para 2º EM. Inclui: definição, coeficientes a/b/c,
vértice, raízes, gráfico, aplicações reais."

SAÍDA: Resumo + Formulário + Exemplos Resolvidos
```

### TERÇA — Aula 1 (Conceituação)
```
"@interativo-mat Cria uma apresentação PPTX interativa
sobre Funções Quadráticas. Inclui simulador onde alunos
movem sliders e veem o gráfico mudar (a, b, c).
Vou usar como abertura para provocar descoberta."

SAÍDA: PPTX com simulador integrado
```

### QUARTA — Aula 2 (Aplicação)
```
"@diferenciado-mat Cria atividade diferenciada sobre
Funções Quadráticas para 2º EM. Preciso de 4 versões
(Abaixo do Básico, Básico, Adequado, Avançado) em
formato de estações. Minha turma tem alunos com
ritmos muito diferentes."

SAÍDA: 4 atividades + Plano de estações + Orientações
```

### QUINTA — Exercícios
```
"@problemista-mate Cria lista de exercícios sobre
Funções Quadráticas para 2º EM. Faço: 20 exercícios
totalizando fixação → reforço → desafio. Incluir
aplicações do ENEM."

SAÍDA: Lista + Gabarito detalhado + Dicas pedagógicas
```

### SEXTA — Avaliação
```
"@avaliador-mat Cria prova sobre Funções Quadráticas
para 2º EM (Rousseau). 6 questões dissertativas,
10 pontos, distribuição Bloom correta. Incluir
questões do FUVEST."

SAÍDA: Prova + Gabarito + Verificação
```

## 7️⃣ Boas Práticas

### ✅ Faça
- Use a equipe de forma integrada (não isolada)
- Comece com SintetiZador para entender o tópico
- Use DiferenciadoMat quando tem turmas heterogêneas
- Sempre termine com AvaliadorMat para validar aprendizado
- Alterne entre agentes conforme a pedagogia (ativo, não passivo)

### ❌ Evite
- Usar só AvaliadorMat (sem exercícios = sem prática)
- Usar só ProblemistaMat (exercícios sem conceitos = sem sentido)
- Ignorar DiferenciadoMat se tem alunos com dificuldades
- Usar tudo ao mesmo tempo sem planejamento
- Esquecer que você é o maestro! Os agentes são instrumentistas
